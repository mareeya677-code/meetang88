# Meetang88 PWA

Progressive Web App สำหรับ Meetang88

## วิธีใช้งาน

1. เปิดเว็บไซต์
2. กดเมนู 3 จุด → "เพิ่มลงในหน้าจอหลัก"
3. กด "เพิ่ม"

## Deploy บน Railway

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new)

1. เชื่อมต่อ GitHub repository นี้
2. Deploy อัตโนมัติ
3. เปิด URL ที่ได้

## ไฟล์สำคัญ

- `index.html` - หน้าเว็บหลัก (wrapper iframe)
- `manifest.json` - ข้อมูล PWA
- `sw.js` - Service Worker
- `icon-192.png`, `icon-512.png` - ไอคอนแอป
